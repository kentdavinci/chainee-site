## Current Content Model

The current version borrows content structure from task-based navigation sites:

- User intent entry points.
- Overseas finance route map.
- Resource and tool library.
- Hong Kong and US investing entry points.
- Crypto and Web3 tool categories.
- Community and action entry.

The visual direction follows the Chainee reference images:

- Monochrome technical editorial style.
- Mixed light and dark sections.
- Node/network visual motif.
- Bilingual brand treatment: 链习社 Chainee.
